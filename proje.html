Medical Prescription System - Details

Key Features

  • Role-Based Access: Secure login for nurses, doctors, and pharmacists.
  • Patient Data Management: Securely stores and retrieves patient information using a unique ID or hospital number.
  • Digital Prescription Module: Doctors can digitally prescribe medication, which is then automatically sent to the pharmacy module.
  • Prescription History: Maintains a clear record of past prescriptions for each patient, enabling doctors to make informed decisions.
  • Automated Billing: Generates a detailed bill for the patient, including all prescribed medications and their costs.
  • Pharmacy Workflow: Allows pharmacists to quickly access a patient's prescription using their ID or phone number, ensuring accurate dispensing of medicines.

Technical Highlights

  • Database Management: Utilized XAMPP to manage the MySQL database, ensuring efficient and reliable data storage.
  • Front-End Development: Developed a dynamic and responsive user interface using Apache NetBeans.
  • JDBC for database connectivity
  • Database Connectivity: Established a robust connection between the Java application and the MySQL database to facilitate seamless data flow.

Development Challenges

Throughout the project, I faced several key development hurdles. The main challenge was integrating the Java front-end with the MySQL database via XAMPP to ensure smooth data flow. Another significant task was creating a secure, role-based user authentication system to maintain data privacy and give nurses, doctors, and pharmacists appropriate access. Finally, I focused on solving the complex issue of data synchronization, ensuring that a doctor's digital prescription was instantly and accurately updated in the pharmacy's module to prevent delays or errors.

Impact

This system significantly improves the traditional medical workflow by digitizing the prescription and dispensing processes, which minimizes errors and speeds up patient service. This leads to better patient safety and increased efficiency for medical staff.

Salon Booking Website - Details

Key Features

  • Easy Appointment Scheduling: Customers can easily view available time slots and book appointments in a few simple clicks.
  • Service and Staff Information: The site provides detailed information about various services and the professionals who offer them.
  • Responsive Design: The website is fully responsive, ensuring a seamless experience on both desktop and mobile devices.

Technical Highlights

  • HTML5 & CSS3: The user interface is built using modern HTML5 and CSS3 standards, including advanced Live CSS techniques for dynamic styling.
  • JavaScript: JavaScript is utilized for interactive elements, form validation, and enhancing overall user experience.

Development Challenges

The primary challenge was ensuring the website's responsiveness, particularly making the booking calendar display clearly and function flawlessly on smaller mobile screens. I focused on optimizing image loading and page speed to enhance the user experience, as a fast-loading site is crucial for customer retention in the service industry. Furthermore, designing an intuitive and simple appointment booking flow was critical, minimizing the steps required to complete a reservation.

Impact

The website provides a professional digital presence for the salon, helping to attract new customers and improve brand image. The online booking system reduces administrative overhead for the staff, allowing them to focus more on clients. It also offers customers the convenience of scheduling appointments 24/7, leading to an increase in overall bookings and customer satisfaction.

SuperMarket Billing System - Details

Key Features

  • Streamlined Inventory Management: Manage products, stock levels, and pricing with a simple, integrated system.
  • Automated Barcode Generation: The system automatically creates a unique barcode for every new product, making billing fast and accurate.
  • Efficient Barcode-Based Billing: Customers' bills are generated quickly and accurately by scanning product barcodes at checkout.
  • Flexible Discount System: Easily apply time-limited discounts to any product, allowing for special promotions and offers.
  • Secure and Reliable Data Storage: All inventory and sales data are securely stored and managed using a MySQL database.
  • Comprehensive Daily Sales Reports: Gain valuable insights into business performance with detailed reports on daily sales and popular products.

Technical Highlights

  • Java and Swing: Used for the robust, cross-platform desktop application interface.
  • MySQL Database: Employed for high-performance and reliable data storage of products and sales records.
  • Barcode Generation API: Integrated a Java library for generating and reading barcodes.
  • Security: Implemented user-role management and secure database connection protocols.

Development Challenges

A major hurdle was implementing the live barcode generation and scanning logic to ensure instantaneous and accurate product lookup, which is critical during checkout. Integrating the discount system smoothly with the billing module without causing transaction errors required careful attention to database logic. I also concentrated on developing a user-friendly and intuitive interface that minimizes the learning curve for new supermarket staff.

Impact

The system enhances operational efficiency by automating inventory updates and billing, reducing manual errors. This translates to a better customer experience with quicker checkout times and reliable service. Furthermore, the detailed daily reports provide business owners with valuable insights into their sales and inventory, enabling them to make smarter, data-driven decisions.

Weladapola.lk - Details

Key Features

  • User Registration & Login System: Allows users to create accounts and securely log in to access platform features.
  • Free Item Marketplace: Registered users can post items they want to give away for free.
  • Item Listing Management: Users can create, edit, and delete their own listings with descriptions and images.
  • Community-Driven Platform: Fosters a community where people can share items they no longer need.
  • Responsive Design: Optimized for all devices from desktop to mobile.

Technical Highlights

  • Backend Development: PHP was used for server-side scripting, handling user authentication and database interactions.
  • Database Management: Utilized MySQL to manage the users and item listings data efficiently.
  • Front-End: Built with a combination of HTML, CSS, and vanilla JavaScript for a dynamic and responsive user experience.
  • Image Handling: Implemented secure file upload functionality for item images.

Development Challenges

Securing the platform was a critical challenge, requiring robust validation and sanitization for all user inputs, especially for listing details and image uploads, to prevent XSS and other common web vulnerabilities. I also had to optimize the listing query logic to ensure fast search results, even as the number of user posts grew. Managing the dynamic state of item availability and user interaction required careful planning of the application logic.

Impact

Weladapola.lk encourages community members to share items they no longer need rather than discarding them. This promotes sustainability and reduces waste while helping community members acquire items they need at no cost. The platform fosters local connections and creates a culture of sharing within the community, demonstrating how technology can be used to build meaningful social connections and promote environmentally conscious practices.

Cricket Blog - Details

Key Features

  • User Authentication System: Secure login and registration system for cricket enthusiasts
  • Blog Creation & Publishing: Registered users can write and publish cricket-related articles and analyses
  • Interactive Commenting System: Users can comment on blog posts to share their thoughts and opinions
  • Public Content Access: Anyone can read blog posts without requiring an account
  • Content Management: Users can manage their own blog posts and comments
  • Cricket-Focused Community: Dedicated platform for cricket discussions and content sharing

Technical Highlights

  • Backend: PHP for server-side logic and processing forms and user data.
  • Database: MySQL for managing blog posts, user information, and comments.
  • Front-End: HTML5, CSS3, and JavaScript for a dynamic and user-friendly interface.
  • Security: Implemented best practices for session management and form security.

Development Challenges

Building a secure and scalable user authentication system was paramount to protect user data and content integrity. The content editor needed to be both powerful enough for rich-text article creation and simple for all users. Optimizing the database to ensure fast loading times for the main blog feed and individual posts while providing rich functionality was also a key consideration. Implementing a comment system that was both user-friendly and protected against spam required thoughtful design and validation mechanisms.

Impact

The Cricket Blog platform creates a dedicated space for cricket enthusiasts to share their knowledge, analyses, and opinions about the sport. By allowing users to both read and contribute content, it fosters an engaged community of cricket fans. The platform encourages knowledge sharing and discussion about cricket matches, players, and strategies. It provides a valuable resource for cricket fans to stay informed and connected with the latest developments in the sport, while also giving aspiring cricket writers a platform to showcase their expertise and writing skills.
